velm

This module contains VisualElment type and some static methods.

velm()

velm(elm)

Similar to new velm.VisualElement(elm).

.batch()

velm.batch(nodes,action,args).

Perform the specified action for each of the specified elements.

Return value

the velm module object for chains call.

Parameter

  • nodes
    TYPE: ArrayLike
    the target dom elements.
  • action
    TYPE: String
    the action name to be performed ,such as “addClass”
  • args
    TYPE: ArrayLike
    the parameters for action to performed

.delegate()

velm.delegate(names,context)

Extend VisualElement prototype with wrapping the specified methods.

Return value

the velm module object for chains call.

Parameter

  • names
    TYPE: ArrayLike
    the methods to be wrapped.
  • context
    TYPE: Object
    the provider of methods wrapped

.find()

velm.find(selector)

find a dom element matched by the specified selector.

Return value

a VisualElemet object wrapping matched dom element, or null if not found.

Parameter

  • selector
    TYPE: String
    a selector expression to match elements against.

.root

velm.root

the VisualElement object wrapping document.body

.VisuleElement

velm.VisualElement

VisualElement is a skylark class type wrapping a visule dom node,provides a number of prototype methods and supports chain calls.

constructor

var ve = new VisualElement(elm);

Return value

The VisualElement object.

Parameter

  • elm
    TYPE : HTMLElement or String
    the wrapped dom element or it's id.

prototype attribute

  • domNode
    TYPE：HTMLElement
    the wrapped dom element
