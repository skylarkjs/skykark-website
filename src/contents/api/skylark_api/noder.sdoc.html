throb()

noder.throb(elm, options)

Add a throbber animation into the specified element.

Return value

An object that contains the remove() method is used to remvoe the throbber animation.

Parameter

  • elm
    TYPE : HTMLElement
    the target dom element
  • setting
    An Object contains params to control throbber animation.
    • callback
      TYPE : Function
      The callback function when the animation is started
    • className
      TYPE : String
      The class name of throbber container div.
    • style
      TYPE : PlainObject
      The styles of throbber container div.
    • time
      TYPE : Number
      A number in milliseconds determining how long the animation is delayed.
