datax

This module contains some methods related to attribute and property of dom node.

.attr()

datax.attr(elm, name, value)

Get or set the value of an attribute for the specified element.

Return value

the attribute value that was set when getting or the datax module object for chains call on setting.

Parameter

  • elm
    TYPE : HTMLElement
    The target dom element.
  • name
    TYPE : String
    The name of the attribute
  • value
    TYPE : Any
    The new attribute value. If null, the specified attribute will be removed (as in .removeAttr())
datax.attr(elm, attrs)

set one or more attributes for the specified element.

Return value

the datax module object for chains call

Parameter

  • elm
    TYPE : HTMLElement
    the target dom element
  • attrs
    TYPE : PlainObject
    An object of attribute-value pairs to set.

.data()

datax.data(elm, name, value)

Store arbitrary data associated with the specified element and/or return the value that was set.

Return value

the data value that was set when getting or the datax module object for chains call on setting.

Parameter

  • elm
    TYPE : HTMLElement
    the target dom element
  • name
    TYPE : String
    The name for the piece of data.
  • value
    TYPE : Any
    The new data value.

.text()

datax.text(elm,txt)

Get or set the text contents of the specified element.

  • elm
    TYPE : HTMLElement
    the target dom element
  • txt
    TYPE: String or Number or Boolean
    The text to set as the content of each matched element. When Number or Boolean is supplied, it will be converted to a String representation

.val()

datax.val(elm, value)

Get or set the current value of the specified element.

Return value

Parameter

  • elm
    TYPE : HTMLElement
    the target dom element
  • value
    TYPE: String or Number or Array
    A string of text, a number, or an array of strings corresponding to the value of each matched element to set as selected/checked
