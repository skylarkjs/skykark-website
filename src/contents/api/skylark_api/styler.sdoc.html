styler

These methods get and set style-related properties of element.

.addClass()

styler.addClass(elm,className)

Adds the specified class(es) to each element in the set of matched elements.

Return value

the styler module object for chains call.

Parameter

  • elm
    TYPE : HTMLElement
    the target dom element
  • className
    TYPE: String
    One or more space-separated classes to be added to the class attribute of the target element

className()

styler.className(elm, value)

Get or set the value of class attribute for the specified element.

Return value

the class attribute value that was set when getting or the styler module object for chains call on setting.

Parameter

  • elm
    TYPE : HTMLElement
    the target dom element
  • value
    TYPE: String
    The new class attribute value.

css()

styler.css(elm, property, value)

Get the value of a computed style property for the first element in the set of matched elements or set one or more CSS properties for every matched element.

Return value

the css attribute value when getting or the styler module object for chains call on setting.

Parameter

  • elm
    TYPE : HTMLElement
    the target dom element
  • property
    TYPE : String
    The name of the attribute
  • value
    TYPE : Any
    The new attribute value. If null, the specified attribute will be removed (as in .removeAttr())

hasClass()

styler.hasClass(elm,name)

Determine whether any of the matched elements are assigned the given class.

Return value

true of false.

Parameter

  • elm
    TYPE : HTMLElement
    the target dom element
  • name
    TYPE : String
    The class name to search for

removeClass()

styler.removeClass(elm,name)

Remove a single class, multiple classes, or all classes from each element in the set of matched elements.

Parameter

toggleClass()

styler.toggleClass(elm, name, when)

Add or remove one or more classes from the specified element.

Return value

the styler module object for chains call.

Parameter

  • elm
    TYPE : HTMLElement
    the target element
  • name
    Type: String
    One or more class names (separated by spaces) to be toggled for each element in the matched set
  • when
