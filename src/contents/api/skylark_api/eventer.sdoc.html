eventer

create()

eventer.create(type,props)

Creates an event object of the specified type.

Return value

The event object.

Parameter

  • type
    TYPE : String
    the event type, a native DOM event type or a custom event type.
  • props
    TYPE : PlainObject Optional
    An object of name-value pairs to set event property.

off()

eventer.off(elm, events, selector, callback)

Remove an event handler for one or more events from the specified element.

Return value

the eventer module object for chains call.

Parameter

  • elm
    TYPE : HTMLElement
    the target dom element
  • events
    TYPE: String
    One or more space-separated event types and optional namespaces, such as “click” or “keydown.myPlugin”.
  • selector
    TYPE: String　Optional
    A selector string to filter the descendants of the specified element that trigger the event. If the selector is null or omitted, the event is always triggered when it reaches the specified element.
  • callback
    TYPE: Function
    The callback function

on()

eventer.on(elm, events, selector, data, callback, one)

Attach an event handler function for one or more events to the selected elements.

Return value

the eventer module object for chains call.

Parameter

  • elm
    TYPE : HTMLElement
    the target dom element
  • events
    TYPE: String
    One or more space-separated event types and optional namespaces, such as “click” or “keydown.myPlugin”.
  • selector
    TYPE: String　Optional
    A selector string to filter the descendants of the specified element that trigger the event. If the selector is null or omitted, the event is always triggered when it reaches the specified element.
  • data
    TYPE: Anything　Optional
    Data to be passed to the handler in event.data when an event is triggered.
  • callback
    TYPE: Function
    The callback function
  • one
    TYPE: Boolean Optional
    If false then the callback function is executed only once.

shortcuts()

eventer.shortcuts(elm)

Return a handler object for the specified element to add shortcut key.

Return value

a handler object contains the following methods:

  • add(pattern, fn)
    Add a callback function on the specified shortcut key.
    • pattern
      TYPE : String
      the shortcut key string ,such as : alt+X, alt+ctrl+shift+Z
    • fn
      TYPE : Function
      the callback function

Parameter

  • elm
    TYPE : HTMLElement
    the target dom element

stop()

eventer.stop(event)

Prevents propagation and clobbers the default action of the passed event. The same as calling event.preventDefault() and event.stopPropagation().

Return value

the eventer module object for chains call.

Parameter

  • event
    TYPE : Event
    the event object to be stopped.

trigger()

eventer.trigger(evented, type, args)

Execute all handlers and behaviors attached to the matched elements for the given event type.

Return value

the eventer module object for chains call.

Parameter

  • evented
    TYPE: Element or Eventted
    the target object, a dom element or Evented obeject.
  • type
    TYPE: String
    A string containing a JavaScript event type, such as click or submit.
  • args
    TYPE: Array or PlainObject
    Additional parameters to pass along to the event handler.
