css

This module contains some methods related to stylesheet and css rule.

.createStyleSheet()

css.createStyleSheet(cssText)

Create a stylesheet with css text and add to the document.

Return value

An id identifing the created stylesheet.

Parameter

  • cssText
    TYPE : String
    The css text of the stylesheet.

.loadStyleSheet()

css.loadStyleSheet(url, loadedCallback, errorCallback)

Create a stylesheet from a url and add to the document.

Return value

An id identifing the created stylesheet.

Parameter

  • url
    The stylesheet file url.
  • loadedCallback
    A callback function when the stylesheet is loaded successful.
  • errorCallback
    A callback function when the stylesheet fails to load.

.deleteSheetRule()

css.deleteSheetRule(sheetId, rule)

Remove a rule from the specified stylesheet.

Return value

The css module object for chains call.

Parameter

  • sheetId
    TYPE: Number
    Id identifing the target stylesheet.
  • rule
    TYPE: Number or CSSRule
    The rule to be removed.

.deleteRule()

css.deleteRule(rule)

Remove a rule from the default stylesheet.

Return value

The css module object for chains call.

Parameter

  • rule
    TYPE: Number or CSSRule
    The rule to be removed.

.removeStyleSheet()

css.removeStyleSheet(sheetId)

Remove the specified stylesheet from the document.

Return value

The css module object for chains call.

Parameter

  • sheetId
    TYPE: Number
    Id identifing the target stylesheet.

  • index
    TYPE: Number
