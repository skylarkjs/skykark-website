dnd

draggable()

dnd.draggable(elm, params)

Make the specified element be in a moveable state.

Return value

A Draggable object

Parameter

  • elm
    TYPE : HTMLElement
    the target dom element
  • params
    An Object contains params to control elmement movement.
    • handle
      TYPE : HTMLElement
      The handle element that allow the move operation, default is elm.
    • constraints
      TYPE : { minX: value1,minY:value2,maxX: vaule3 ,maxY: value4}
      This parameter is used to limit the range of element movement
    • started
      TYPE : Function
      The callback function when the movement is started
    • moving
      TYPE : Function
      The callback function when moving
    • stopped
      TYPE : Function
      The callback function when the movement is stopped
