filer

dropzone()

filer.dropzone(elm, params)

Make the specified element to could accept HTML5 file drag and drop.

Return value

the filer module object for chains call.

Parameter

  • elm
    TYPE : HTMLElement
    the target dom element
  • params
    TYPE : PlainObject
    An Object contains params to control file picker.
    • droped
      TYPE : Function
      The callback function when the files is droped.
      synax : function droped(files) files is a list of File objects

picker()

filer.picker(elm, params)

Make the specified element to pop-up the file selection dialog box when clicked , and read the contents the files selected from client file system by user.

Return value

the filer module object for chains call.

Parameter

  • elm
    TYPE : HTMLElement
    the target dom element
  • params
    TYPE : PlainObject
    An Object contains params to control file picker.
    • picked
      TYPE : Function
      The callback function when the files is selected.
      synax : function picked(files) files is a list of File objects
