langx

This module contains some methods to extend javascript language function.

.all()

langx.all(promises)

This method can be useful for aggregating the results of multiple promises.

Return value

  • An already resolved Promise if the An already resolved Promise if the promises passed is empty or contains no promises.
  • A pending Promise in all other cases. This returned promise is then resolved/rejected langxhronously (as soon as the stack is empty) when all the promises in the given promises have resolved, or if any of the promises reject.

Parameter

  • promises
    TYPE : Array
    Zero or more Promise objects.

Return value

Parameter

    TYPE : Function
    The function Will be executed by the function.
  • wait
    TYPE : Number
    The delay executed time in milliseconds.

Deferred

var d = new Deferred();

Deferred is a class that is used as the foundation for managing langxhronous threads. use this class to get a promise that gets resolved when the langxhronous thread is complete. In order trigger a callback to occur when the thread is complete, the d.promise.then() method is used.

Deferred.prototype.promise

Points to the promise object related to this deferred object.

Deferred.prototype.resolve

resolve(value)

Resolve this deferred object.

When the deferred is resolved, any onResolved function added by promise.then() are called. Callbacks are executed in the order they were added. Each callback is passed the value argument.

Deferred.prototype.reject

reject(reason)

Reject this deferred object.

When the deferred is rejected, any onRejected function added by promise.then() are called. Callbacks are executed in the order they were added. Each callback is passed the reason argument.

.first()

langx.first(promises)

This method returns a promise that resolves or rejects as soon as one of the promises in the promises resolves or rejects, with the value or reason from that promise.

Return value

A pending Promise that resolves or rejects langxhronically (as soon as the stack is empty) as soon as one of the promises in the given promises resolves or rejects, adopting that first promise's value as its value.

Parameter

  • promises
    TYPE : Array
    Promise objects

.klass()

langx.klass(props, parent, options)

Create a new skylark class type.

Return value

the new class type created.

every skylark class type contains the following static methods:

  • partial(props,options)
    add new properties into the class type
  • inherit(props,options)
    create a new class type inherited from the class type

Parameter

  • props
    TYPE: PlainObject
    the properties of the class to be created, contains :
    • klassName
      TYPE: String
      the name of the class to be created
    • init
      TYPE: String
      the initialize method of the class to be created
    • other properties and methods
  • parent
    TYPE: Skylark Class Optional
    the parent class of the class to be created
  • options
    TYPE: PlainObject Optional
    the options for creating class, contains:
    • noOverrided
      TYPE: Boolean
      if true, not support calling this.overrided() to execute the same method of the parent class

.when()

langx.when(valueOrPromise,callback，errback，progback)

This method transparently applies callbacks to values or promise.

Return value

the langx module object called.

Parameter

  • valueOrPromise
    TYPE : Object
    Either a value or a promise.
  • callback
    TYPE : Function
    The function that should be called back when the promise is fulfilled, or will be called immediately if valueOrPromise is not a promise.
  • errback
    TYPE : Function Optional
    The function that should be called when a promise is rejected.
  • progback
    TYPE : Function Optional
    The function that should be called when a promise provides progress.
