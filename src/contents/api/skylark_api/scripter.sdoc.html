scripter

loadJavaScript()

scripter.loadJavaScript(url, loadedCallback, errorCallback)

Load a script from a url into the document.

Return value

An id identifing the loaded script.

Parameter

  • url
    The javascript file url.
  • loadedCallback
    A callback function when the script is loaded successful.
  • errorCallback
    A callback function when the script fails to load.

deleteJavaScript()

scripter.deleteJavaScript(id)

Remove the specified script from the document.

Return value

The scripter module object for chains call.

Parameter

  • id
    TYPE: Number
    Id identifing the target script .
