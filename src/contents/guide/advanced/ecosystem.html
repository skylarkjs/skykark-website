Ecosystem

skylark.js is a highly modularized ecosystem that constitutes a variety of modules that are inherently independent and can be used independently.

Constitution

The following table lists the main components of the skylark.js ecosystem:

Project Description Integration
skylark-langx (github) javaScript language extension library built-in on skylark.js runtime
skylark-router (github) frontend routing framework built-in on skylark.js runtime
skylark-spa (github) html5 single page application framework built-in on skylark.js runtime
skylark-utils (github)   javascript dom utility library built-in on skylark.js runtime(full build version)
skylark-slax-browser (github) official desktop browser for running skylark.js application built-in on skylark.js development tools
skylark-slax-nodeserver (github) official node server for running the skylark.js application built-in on skylark.js development tools
[skylark-jquery] (github) an extension library fully compatible with jquery api optional, installed by the developer as needed
[skylark-backbone] (github) an extension library fully compatible with backbone api optional, installed by the developer as needed

skylark-langx

Introduction

A simple JavaScript language extension library, including class support, Evented class, Deferred class and some commonly used tool functions.

Integration

This library is built-in on skylark.js core library as skylarkjs.langx.

skylark-utils

Introduction

skylark-utils is a powerful and concise dom utility library.

Integration

This library is built-in on skylark.js library,the correspondence between each sub-module is as follows

skylarkjs.browser skylark-utils/browser
skylarkjs.css skylark-utils/css

skylark-router

Introduction

skylark-router is an elaborate front-end routing framework.

Integration

This library is completely built-in on skylark.js, and when you use skylark.js to develop an application, you do not have to use this library's API directly.

skylark-spa

Introduction

skylark-spa is an Elegant HTML5 Single Page Application Framework.

Integration

This library is completely built-in on skylark.js, and when you use skylark.js to develop an application, you do not have to use this library's API directly.

skylark.js will automatically create an instance of spa.Application after startup and start the application.
