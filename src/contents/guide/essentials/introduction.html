Introduction

Skylark.js is an unified framework for building modern html5 applications across web and desktop, including the runtime library and many development tools.

Runtime

different builds

build Description
full skylarkjs.js included dom utilities
core skylarkjs-core.js not included dom utilities
full （development） uncompressed/skylarkjs.js included dom utilities
core （development） uncompressed/skylarkjs-core.js not included dom utilities

installation

There are multiple ways to install the skylark.js runtime library.

  1. cdn
    http://registry.skylarkjs.org/packages/skylarkjs/v0.9.3/skylarkjs.js or
    http://registry.skylarkjs.org/packages/skylarkjs/v0.9.3/uncompressed/skylarkjs.js
    0.9.3 is the version number, please replace to the required version, usually the latest version.
  2. bower
    bower install skylarkjs
  3. sdk tool
    When you create a project skeleton with the SDK tool, the runtime library is automatically copied to the project's lib directory.

usage

Using the skylarkjs library for a AMD module. 

<script type="text/javascript"
            src="https://cdnjs.cloudflare.com/ajax/libs/require.js/2.3.3/require.min.js"
            data-main="http://registry.skylarkjs.org/packages/skylarkjs/v0.9.3/uncompressed/skylarkjs.js">
</script>
<script>
  require(["skylarkjs"],funciton(skylarkjs){
  skylarkjs.xxx();
  });
</script>
