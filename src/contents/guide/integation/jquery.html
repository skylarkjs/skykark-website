jquery

Overview

Jquery is a most popular javascript library, with a large number of plug-ins and application using jquery.
The optional skylark.jquery library is based on the skylark/query module, providing APIs that are fully compatible with jquery,and the codes is simpler and more productive, JQuery plugins and applications can run directly on skylark without jquery.

Installation

In order to use skylark-jquery library, there are two options of using CDN directly or copying the library to the lib folder of the project using bower or npm.
Then setup the skylark-jquery library in slax-config.json file.

using cdn : 

{
    "runtime" : {
      "skylarkjs" : {
        "version" : "0.9.3"
      },
      "paths": {
        "jquery" : "http://registry.skylarkjs.org/packages/skylark-jquery/v0.9.3/uncompressed/skylark-jquery"
      }
    }
}

using lib copy: 

{
    "runtime" : {
      "skylarkjs" : {
        "version" : "0.9.3"
      },
      "paths": {
        "jquery" : "./lib/skylark-jquery/uncompressed/skylark-jquery"
      }
    }
 }

Usage

use skylark-jquery

the method is exactly the same as using original jquery. 

define(["jquery"],function($) {
  $(".btn").button();
});

integrate existed jquery plugin

To use bootstrap as an example, configed as follows: 

{
    "runtime" : {
      "skylarkjs" : {
        "version" : "0.9.3"
      },
      "shim": {
          "bootstrap": {
              "deps": ["jquery"],
              "exports": "bootstrap"
          }
      },
      "paths": {
        "jquery" : "./lib/skylark-jquery/uncompressed/skylark-jquery",
        "bootstrap" : "https://maxcdn.bootstrapcdn.com/bootstrap/3.3.2/js/bootstrap.min",
      }
    }
 }
